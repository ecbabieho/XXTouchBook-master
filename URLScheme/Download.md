## 下载相关

```
xxt://download/?path=%@&url=%@
```

从指定 URL 下载文件. ```path``` 为欲保存的文件名, ```url``` 为指定的 URL.

- 说明
    - 如果不指定 ```path```, 文件名将自动从目标 URL 猜测, 或选取 URL 的最后一部分作为文件名.
    - ```path``` 可指定为绝对路径或相对路径, 若指定为相对路径, 则其路径是相对于 App 文件浏览器当前所在的目录而言的.


- 二维码字典

```json
{
    "event":"download",
    "path":"[需要保存的文件名.lua]",
    "url":"[下载地址]"
}
```
