### \! 建立一个对话框对象 \(**dialog**\)


#### 声明
```lua
对话框对象 = dialog()
```


#### 参数及返回值
- 对话框对象
    - 对话框，返回一个对话框对象


#### 说明
> 建立一个对话框对象  
> **注意** 这个函数没有参数，**请不要** 给任何参数，带参数调用是弹出一个弹窗，声明如下  
```lua
dialog(弹窗内容:文本型, 超时秒:实数型)
```
> 例如  
```lua
dialog('Hello, XXTouch!', 10)
```


#### 示例  
```lua
dialog():show()
```
