### 对 base64 编码的文本进行解码 \(**string\.base64\_decode**\)


#### 声明
```lua
数据内容 = string.base64_decode(b64文本)
```


#### 参数及返回值
- b64文本
    - 文本型，base64 编码的文本
- 数据内容
    - 字符串型，返回解码后的字符串


#### 说明
> string\.base64\_encode 的反函数，将 base64 编码的文本转换回字符串 (或二进制数据块)
